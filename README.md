# Bank Transaction Anomaly Detection

This project detects anomalies in bank transactions using statistical analysis techniques like Z-score.

## Features
- Detects unusual transactions
- Uses mean and standard deviation
- Visualizes data using graphs

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib

## Files
- BankTransactionAnomalyDetection.py
- bank_transactions.csv

## How to Run
1. Install required libraries
2. Run the Python file
3. View output and graphs

## Output
- Histogram
- Box Plot
- Scatter Plot
- Anomaly Detection Results
